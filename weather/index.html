Site Purpose

The purpose of this site is to provide information on the weather in any part of the world based on user location or based on location search. The site will be user friendly, pleasing to the eye and provide adequate information.

Site Name

The site name will be "Primetime Cloud 9" and the site domain will be primetimecloud9.com.

Site Logo

The logo is a visual representaiton of the site name. It includes a cloud image with the number 9 to add a visual compliment to the site name and site overall.
